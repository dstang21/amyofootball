# Chat Images

This folder contains images that can be displayed in chat messages.

## Usage

To display an image in chat, type `/pic/filename` where `filename` is the name of the image file (without the .png extension).

For example:
- `/pic/touchdown` will display `touchdown.png`
- `/pic/celebration` will display `celebration.png`
- `/pic/sad` will display `sad.png`

## Adding Images

1. Save your image as a PNG file in this directory
2. Use lowercase letters, numbers, hyphens, or underscores in the filename
3. Reference it in chat using `/pic/your-filename`

## Image Guidelines

- Images should be in PNG format
- Recommended max size: 800x800 pixels
- Images will be automatically scaled to max 300x300 in chat display
